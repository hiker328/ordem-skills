# Exemplos

Amostras **fictícias** (dados inventados) para mostrar o que cada skill recebe e o
que entrega. Servem de referência de formato — para você e para o agente.

| Arquivo | O que é |
|---------|---------|
| `conversa-exemplo.txt` | Conversa de WhatsApp de entrada (formato que `evolution_fetch.py` gera) |
| `analise-exemplo.md` | Saída da `analisar-conversas-closer` sobre a conversa acima |
| `qualificacao-exemplo.md` | Saída da `qualificar-lead` sobre a mesma conversa |
| `followup-exemplo.md` | Saída da `gerar-followup` (sequência de cadência) |
| `notas-crm-exemplo.md` | Saída da `resumir-call-para-crm` a partir de uma call |

> Nenhum dado aqui é real. Nomes, números e empresas são inventados.
